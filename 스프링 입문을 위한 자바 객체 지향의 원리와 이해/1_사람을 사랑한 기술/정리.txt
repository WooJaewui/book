 # 기계어에서 객체 지향 프로그래밍 언어로.

컴퓨터.
- 0과 1만 아는 바보.
- 하지만, 매우 빠르고 정확하다. ( 사람과 다르게 실수가 없다 )

니모닉 Mnemonic.
- 어떤 것을 기억하는 데 쉽게 하도록 도움을 주는 것. 또는 쉽게 기억되는 성질.

기계어 특징.
- 0과 1의 나열.
- 기종마다 하나씩 소스파일 생성.
- 목적 파일이 소스 그 자체.

어셈블리어.
- 일상 단어 사용.
- 기종마다 하나씩 소스파일 생성.
- 목적 파일은 어셈블러로 소스를 번역해 생성.
- 기계어와 1;1 대응하는 니모닉.

C언어.
- 수학적 기호 사용.
- 기종이 몇 개든 단 하나.
- 목적 파일은 컴파일러로 소스를 법ㄴ역해 생성.
- 기계어와 m : n 대응하는 수학적 기호.
