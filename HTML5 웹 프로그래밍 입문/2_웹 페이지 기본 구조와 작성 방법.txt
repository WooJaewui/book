
 웹 페이지 기본 구조와 작성 방법.



 # 태그와 요소.

요소 Element.
HTML 페이지를 구성하는 각 부품.

태그 Tag.
요소를 만들 때 사용하는 작성 방법.
사용할 수 있는 태그는 모두 W3C ( World Wide Web Consortium )에서 지정해 놓았다.
요소와 태그를 많이 구분하지 않고 같은 뜻으로 사용한다.



 # 생성 방법에 따른 태그 구분.

내용을 가질 수 있는 태그.
<태그 이름> 내용 </태그 이름>

종류.
<h1>text</h1> , <p>hello</p>

내용을 가질 수 없는 태그.
<태그 이름>

종류.
<img>, <hr>, <br>



 # 속성.

속성 Attribute.
태그에 추가 정보를 부여할 때 사용하는 것이다.

사용.
<img src="image.png">



 # 주석.

주석.
어떤 기능을 하는 코드인지 설명을 기록하는 방법이다.

사용.
<!- 주석 내용 -->



 # head 태그 내부에 넣을 수 있는 태그.

종류.
<meta> : 웹 페이지에 추가 정보 전달.
<title> : 페이지 제목 지정.
<script> : 웹 페이지에 스크립트 추가.
<link> : 웹 페이지에 다른 파일 추가.
<style> : 웹 페이지에 스타일시트 추가.
<base> : 웹 페이지의 기본 경로 지정.



 # css 사용하는 방법.

html 내부에서 작성.
<head>
<style>
	css 언어 ;
</style>
</head>

외부에서 작성.
파일이름.css 파일 생성.
파일이름.css 파일 안에 실행할 내용 작성.
html에서 <link rel="stylesheet" href="파일이름.css">



 # Javascript 사용하는 방법.

html 내부에서 작성.
<head>
<script>
	JS 언어  ;
</script>
</head>

외부에서 작성.
파일이름.js 파일 생성.
파일이름.js 파일 안에 실행할 내용 작성.
html에서 <script src="파일이름.js"></script>

용어.
