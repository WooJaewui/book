 # 배포 지원을 위한 FrontPage 서버 확장.

FP FrontPage.
다양한 기능을 제공하는 마이크로소프트사의 웹 개발 및 배포 도구 집합.

FPSE FrontPage Server Extensions.
'어디서든 배포한다' 라는 전략의 하나로 서버 측 소프트웨어 제품군을 출시했다.
FrontPage 배포 프로토콜은 HTTP POST 요청 위에 RPC 계층을 구현했다.

RPC Remote Procedure Call.
별도의 원격 제어를 위한 코딩 없이 다른 주소 공간에서 원격 제어를 할 수 있는 통신방법이다.

가상 서버.
웹 서버 한 개에서 웹 사이트 여러 개를 호스팅할 수 있다.
가상 서버를 지원하는 웹 서버를 다중 호스팅 웹 서버라 부른다.

루트 웹.
웹 서버의 최사우이 컨텐츠 디렉토리를 의미한다.
웹 서버에서 오직 한 개의 루트 웹만 존재한다.

서브 웹.
루트 웹의 하위디렉토리거나 완전한 FPSE 확장 웹인 다른 서브 웹의 하위 디렉토리이다.

IIS Internet Information Services.
마이크로소프트 윈도우를 사용하는 서버들을 위한 인터넷 기반 서비스들의 모임이다.

접근 사용자.
아파치, NCSA 웹 서버 - .htaccess에 명시한다.
넷스케이프 - .nsconfig에 명시한다.



 # WebDAV와 공동 저작.

WebDAV Web Distributed Authoring and Versioning.
웹 분산 저작 및 버전 관리.

WebDAV 메소드.
새로운 HTTP 메소드 집합을 정의하고 몇몇 기존 메소드를 수정하였다.

WebDAV에서 XML사용 용도.
데이터를 어떻게 처리할 것인지 설명하는 명령 포맷.
서버의 복잡한 응답을ㅈ 표현하는 데 사용하는 포맷.
콜렉션과 리소스를 처리하는 데 사용하는 커스텀 정보 포맷.
데이터 자체를 표현할 수 있는 유연한 포맷.
대부분의 국제화 관련 문제에 대한 훌륭한 해결책.

DTD Document Type Definition.
문서 타입을 정의.

namespace.
xml namespace( 이름공간 )은 요소나 속성 이름의 집합이다.
