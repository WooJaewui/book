 # JVM T자형.

스태틱 area.
- 클래스, 스태틱 멤버, lang 패키지 등이 존재한다.

스택 area.
- 메소드가 수행되는 스택 영역.
- 기본형 변수는 값을 가지고 있고, 참조형 변수는 주소값을 가지고 있다.

힙 area.
- 생성된 객체를 저장하는 공간.
- 자식 객체를 생성하면 부모 객체는 자동으로 생성된다.



 # 캡상추다.

추상화.
- 추상화는 모델링이다.
- 객체란 현실세계에 있는 물체를 코드로 구현하는 것이다.
- 객체를 구현할 때 필요한 상태(변수)들만 가져와서 모델링하는 것을 추상화라고 한다.

상속.
- 상속은 확장이다.
- 하위클래스 is a kind of 상위 클래스이다.
- 상속은 많을 수록 좋고, 인터페이스는 적을 수록 좋다. ( 뒤에서 자세히 배움 )

다형성.
- 다형성은 부모 타입의 변수에 자식 인스턴스를 대입해서 사용하는 것이다.
- 핵심은 override이다. ( overload도 포함하는 경우도 있다 )
