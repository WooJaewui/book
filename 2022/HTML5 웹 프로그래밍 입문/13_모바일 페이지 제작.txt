 # 동위 선택자.

선택자.
선택자A + 선택자B : 선택자A 바로 뒤에 위치하는 선택자B를 선택.
선택자A ~ 선택자B : 선택자A 뒤에 오는 모든 선택자B를 선택.

동위 선택자 응용.
#checkbox: checked + nav { display : non; } 
체크박스를 체크하면 어떤 문장이 보이지 않게 설정할 수 있다.

padding 요소의 갯수.
padding 1 2 3 4 = 위, 오른쪽, 아래, 왼쪽.
padding a b c = 위 , 오른쪽 왼쪽 , 아래.
padding a b = 위, 아래
