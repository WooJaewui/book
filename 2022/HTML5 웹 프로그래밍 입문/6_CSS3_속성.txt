
 CSS 속성.



 # 박스 속성.

속성.
margin : 테두리와 다른 태그 사이의 테두리 바깥쪽 여백.
border : 테두리.
padding : 테두리와 글자 사이의 안쪽 여백, 배경색은 padding까지 적용.
width : 글자를 감싸는 영역의 가로 크기.
height : 글자를 감싸는 영역의 세로 크기.

속성 값 입력.
top - bottm - left - right 순으로 값을 넣어서 사용할 수 있다.
padding : 10px 20px 30px 40px ;
2개씩 묶어서 사용 가능. ( 위 아래 ) ( 왼쪽 오른쪽 ) 
border : 10px 30px ;

세부 설정.
border-width : thick = 두께를 두껍게 만들어준다.
border-style : dashed = 점선을 생성한다.
border-color : black = 선을 검정색으로 생성한다.
border-radius : 20px = 모서리에 라운드 처리를 해준다.



 # 가시 속성.

설명.
태그가 화면에 보이는 방식을 지정하는 것. 대표적인 예로 display가 있다.

display 값.
display : none = 태그가 화면에서 보이지 않는다.
display : block = 태그의 속성을 block으로 바꿔준다.
display : inline = 태그의 속성을 inline으로 바꿔준다. ( width, height, margin 등 사용 불가 )
display : inline-block = 태그의 속성을 둘 다 사용가능하도록 바꿔준다. ( width, height, margin 등 사용 가능 )



 # 배경 속성.

background 속성.
background-color : 배경 색상 선택.
background-image : 배경 이미지 삽입.
background-size : 배경 이미지의 크기 지정.
background-repeat : 배경 이미지의 반복 형태 지정.
background-attachment : 배경 이미지의 부착 형태 지정. ( scroll, fixed 등 )
background-position : 배경 이미지의 위치 지정.
background : 한 번에 모든 배경 속성 입력.



 # 글자 속성.

font 속성.
font - size : 글자 크기 설정.
font - family : 컴퓨터의 설치된 글꼴 중에 골라서 선택.
font - style : italic, inherit, initial, normal, oblique 등 글자의 스타일 설정.
font - weight : bold, lighter 등 글자의 두께와 관련된 설정.

text 속성.
text - align : center, end, justify, left 등 글자 정렬을 지정. 
text - decoration : none 등의 세부 사항 설정.

line 속성.
line-height : 70px = 애플리케이션에 많이 사용하는 수직 중앙 정렬.



 # 위치 속성.

position 속성.
position : absolute = 절대 위치 좌표 설정.
position : fixed = 고정 위치 좌표 설정.
position : relative = 초기 위치에서 상하좌우로 위치 이동.
position : static = 위쪽에서 아래쪽으로 순서대로 배치.

overflow 속성.
overflow : hidden = 영역을 벗어나는 부분 감춤.
overflow : scroll = 영역을 벗어나는 부분을 스크롤로 만듦.



 # 유동 속성.

float 속성.
float : left = inline에 사용하면 왼쪽에 딱 붙어서 오른쪽 부분을 사용할 수 있게된다.
float : right = inline에 사용하면 오른쪽에 딱 붙어서 왼쪽 부분을 사용할 수 있게된다.



 # 그림자 속성.

shadow 속성.
text - shadow : 오른쪽 아래 흐림도 색상 = 글자에 그림자를 생성.
box - shadow : 오른쪽 아래 흐림도 색상 = 박스에 그림자를 생성.
