
 자바스크립트 기본 문법.



 # 자바스크립트 기본 용어.

문장과 표현식.
표현식 expression : 값을 만들어 내는 간단한 코드.
문장 statement : 실행할 수 있는 코드의 최소 단위.
문장 마지막에 ; 또는 줄바꿈을 통해 종결을 나타낸다.
표현식에 ; 을 찍으면 문장이 되고, 문장이 모여서 프로그램이 된다.

키워드.
키워드 keyword : JS를 처음 만들 때 정해진 특별한 의미가 부여된 단어.

식별자.
식별자 identifier : 변수나 함수 등에 이름을 붙일 때 사용하는 단어.
클래스 안에 있는 함수를 메소드, 클래스 안에 있는 변수를 속성이라고 부른다.

주석.
주석 : 프로그램 진행에 전혀 영향을 주지 않는 코드, 다른 코드를 설명하는 데 사용한다.
// 주석문 , /* 주석문 */ 등으로 표현한다.



 # 자바 스크립트 출력.

alert.
alert( ) 함수를 사용해 웹 프라우저에 경고 창을 띄우는 것이다.
alert( "메시지" ) ;



 # 자료형.

숫자.
JS에서는 정수와 실수를 구분하지 않는다.
주로 사칙연산을 통해 숫자의 계산이 필요할때 사용한다.

문자열.
문자 집합을 의미한다.
" 메시지 " 와 ' 메시지 ' 두 가지 방법 모두 사용가능하다.
" 메시지 " + " 메시지 " 를 통해 2개의 문자열을 연결할 수 있다.

불
참과 거짓을 표현할 때 사용하는 자료. ( true, false ) 
논리 연산자 ! , && , || 등을 통해 bool 값을 얻을 수 있다.
0 , null , undefined = null이고, 값이 존재하면 true이다.



 # 변수.

변수 정의.
값을 저장할 때 사용하는 식별자이다.

변수 선언.
var a ; let b ; const c ;

변수에 값 할당.
var a = 10 ;
let b = 20 ; 
const c = 30 ; 

변수 초기화.
변수를 선언하고 처음으로 값을 할당하는 것을 의미한다.

변수 사용하기.
var sum = a + b + 10 

주의사항.
변수를 선언하고 값을 할당하지 않으면 undefined가 된다.



 # 배열.

배열.
여러 개의 데이터를 한번에 저장할 수 있는 자료형이다.

배열 선언.
var arr = [ 1,2,3,4,5 ]

배열 사용.
arr[ 0 ] + arr[ 1 ] 

for문을 활용한 배열.
for( var i in 배열이름 ) { alert( 배열이름[ i ] ) } 



 # 조건문.

조건문.
제어문 중 하나로 원하는 조건에 따라 프로그램에 흐름을 제어할 때 사용한다.

if 조건문.
if ( 조건1 ) { 조건1이 true일 때 실행할 문장; }
else if ( 조건2 ) { 조건2가 true일 때 실행할 문장; }
else { 조건이 모두 false일 때 실행할 문장 ; } 



 # 반복문.

반복문.
제어문 중 하나로 원하는 문장을 반복할 때 사용한다.

for문.
for ( var i =0 ; i < 1000 ; i++ ) { 실행할 문장 ; } 

while 반복문.
while( 조건 ) { 조건이 true일 때 동안 반복할 문장 ; } 



 # 함수.

함수.
코드의 집합이다.

익명 함수 선언.
function ( 매개변수 ) { 실행할 문장 ; } 

일반 함수 선언.
function 함수이름 ( 매개변수 ) { 실행할 문장 ; } 

함수 사용.
var 함수이름 = function ( 값 ) { 실행할 문장 ; } 
function 함수이름( 값 ) { 실행할 문장 ; } 
함수이름( ) ;

매개변수.
함수 쪽에 추가적인 정보를 전달하여 함수의 활용성을 높여준다.
function 함수이름 ( x , y ) { x + y ; } 
함수이름( 5, 6 ) = 11

반환값.
함수를 실행한 후에 나오는 결과값.
return 값 ; 으로 표현.

콜백함수.
매개변수로 전달되는 함수를 의미한다.
function 함수이름 ( callback ) { callback( ) ; } 

주의사항.
코드 작성 순서와 관계 없이 일반 함수를 먼저 읽고, 익명 함수를 나중에 읽는다. 



 # 객체.

객체.
자료형 여러 개를 한 번에 저장한다.

객체 선언.
var 객체이름 = { 키1 : 값1, 키2 : 값2, ... } ; 

객체 사용.
객체이름.키1 => 값1을 반환.
객체이름.키2 => 값2를 반환.

for문을 통한 객체 사용.
for ( var i in 객체이름 ) { alert( i  객체이름[ i ] } 

객체 메소드.
var 객체이름 = { 메소드이름 : function ( 매개변수 ) { 실행할내용 ; } } 
객체이름.메소드이름( 매개변수 값 )  => 실행할내용 ;
